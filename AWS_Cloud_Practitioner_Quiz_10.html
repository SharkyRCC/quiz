Question 1

Which AWS service provides a comprehensive view of security posture across AWS accounts?

AWS Security Hub provides a comprehensive view of security posture across AWS accounts by aggregating security findings from various AWS services and third-party tools, providing centralized security findings management and compliance status tracking.

Question 2

Which additional benefit will a company receive with AWS Enterprise Support compared to AWS Business Support?

AWS Enterprise Support provides a designated Technical Account Manager (TAM) who assists in monitoring and optimization. This is a key differentiator from Business Support, providing personalized and proactive support experience.

Question 3

What is a key benefit of AWS's pay-as-you-go pricing model compared to traditional on-premises infrastructure?

AWS's pay-as-you-go model eliminates upfront capital expenditure for hardware and infrastructure. You only pay for resources you actually use, providing significant cost savings and flexibility compared to traditional on-premises infrastructure investments.

Question 4

AWS Fargate is a serverless compute engine for containers that eliminates operational overhead of managing EC2 instances. It reduces unplanned administration costs related to patching, scaling, and maintaining underlying infrastructure while providing serverless container deployment.

Question 5

Which AWS service provides automated backup and point-in-time recovery for databases?

Amazon RDS provides automated backup and point-in-time recovery capabilities for relational databases. It automatically backs up your database and transaction logs, enabling recovery to any point in time within the backup retention period.

Question 6

Which AWS Cloud Adoption Framework (AWS CAF) capabilities belong to the governance perspective? (Choose two)

The governance perspective focuses on ensuring cloud adoption is aligned with business objectives and managed effectively. Program and project management ensures structured approach to cloud initiatives, while risk management identifies and mitigates potential risks.
