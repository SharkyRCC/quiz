Q1 Which AWS service can be used to create and manage APIs?

✓ Correct Answer: B

Amazon API Gateway is a fully managed service that makes it easy for developers to create, publish, maintain, monitor, and secure APIs at any scale.

Q2 Which actions are the responsibility of AWS, according to the AWS shared responsibility model? (Choose two.)

✓ Correct Answer: A,D

AWS is responsible for securing the virtualization layer (hypervisor and underlying hardware) and patching the operating system on managed services like RDS. Customers are responsible for patching EC2 instances and configuring IAM policies.

Q3 What is the AWS service that provides managed threat detection?

✓ Correct Answer: D

All these services provide managed threat detection capabilities: GuardDuty for intelligent threat detection, Inspector for application security assessment, and Security Hub for centralized security findings management.

Q4 What is the AWS service that provides managed business intelligence?

✓ Correct Answer: C

Amazon QuickSight is a scalable, serverless, embeddable, machine learning-powered business intelligence (BI) service built for the cloud.

Q5 Which AWS service provides managed data lake formation?

✓ Correct Answer: B

AWS Lake Formation is a service that makes it easy to set up a secure data lake in days instead of months by simplifying and automating many complex manual steps.

Q6 Which AWS service can be used to monitor and log API calls made to AWS services?

✓ Correct Answer: B

AWS CloudTrail is a service that enables governance, compliance, operational auditing, and risk auditing of your AWS account by logging and monitoring API calls made to AWS services.

Q7 What is the AWS service that provides managed Elasticsearch?

✓ Correct Answer: B

Amazon OpenSearch Service (formerly Amazon Elasticsearch Service) is a managed service that makes it easy to deploy, operate, and scale OpenSearch clusters in the AWS Cloud.

Q8 Which AWS service provides managed ledger database?

✓ Correct Answer: A

Amazon QLDB (Quantum Ledger Database) is a fully managed ledger database that provides a transparent, immutable, and cryptographically verifiable transaction log.

Q9 Which AWS service can be used to create and manage virtual private clouds?

✓ Correct Answer: B

Amazon VPC (Virtual Private Cloud) lets you provision a logically isolated section of the AWS Cloud where you can launch AWS resources in a virtual network that you define.

Q10 Which of the following acts as an instance-level firewall to control inbound and outbound access?

✓ Correct Answer: B

Security groups function as a virtual firewall for your EC2 instances, controlling traffic at the instance level. They operate at the network interface level and are stateful, meaning that if you allow inbound traffic from a source, the return traffic is automatically allowed.

Q11 What is the AWS service that provides managed graph database?

✓ Correct Answer: D

Amazon Neptune is a fast, reliable, fully managed graph database service that makes it easy to build and run applications that work with highly connected datasets.

Q12 Which AWS service provides managed user pools for mobile and web apps?

✓ Correct Answer: B

Amazon Cognito provides user pools that are user directories for mobile and web applications, handling user registration, authentication, account recovery, and other common tasks.

Q13 What is the AWS service that provides managed Apache Airflow?

✓ Correct Answer: B

Amazon MWAA (Managed Workflows for Apache Airflow) is a managed orchestration service for Apache Airflow that makes it easier to set up and operate end-to-end data pipelines in the cloud.

Q14 Which AWS service provides a content delivery network (CDN)?

✓ Correct Answer: B

Amazon CloudFront is a fast content delivery network (CDN) service that securely delivers data, videos, applications, and APIs to customers globally with low latency and high transfer speeds.

Q15 What is the AWS service that provides managed AR/VR content delivery?

✓ Correct Answer: A

Amazon Sumerian was a managed service for creating and running virtual reality (VR), augmented reality (AR), and 3D applications. Note: This service has been discontinued but may still appear in exam questions.

Q16 Which AWS service provides managed configuration management?

✓ Correct Answer: D

Both AWS OpsWorks and AWS Systems Manager provide configuration management capabilities. OpsWorks uses Chef and Puppet, while Systems Manager provides various configuration management tools including State Manager and Parameter Store.

Q17 What is the maximum number of S3 buckets that can be created per AWS account by default?

✓ Correct Answer: B

By default, AWS allows up to 100 S3 buckets per AWS account. This limit can be increased by contacting AWS Support if needed.

Q18 Which AWS service provides a managed NoSQL database?

✓ Correct Answer: C

Amazon DynamoDB is a fully managed NoSQL database service that provides fast and predictable performance with seamless scalability. It handles all the administrative tasks of operating a distributed database.

Q19 Which task can a company perform by using security groups in the AWS Cloud?

✓ Correct Answer: A

Security groups can restrict access based on port number. By configuring security group rules, you can allow only traffic on specific ports (like port 80 for HTTP or port 22 for SSH) to reach your EC2 instance, which is a fundamental security practice.

Q20 Which AWS service provides managed in-memory caching?

✓ Correct Answer: B

Amazon ElastiCache is a web service that makes it easy to deploy, operate, and scale an in-memory cache in the cloud, supporting both Redis and Memcached engines.

Q21 Which AWS service provides managed Apache Spark?

✓ Correct Answer: D

Both Amazon EMR and AWS Glue provide managed Apache Spark capabilities. EMR is a cloud big data platform, while Glue is a serverless data integration service that uses Spark for data processing.

Q22 What is the AWS service that provides managed streaming analytics?

✓ Correct Answer: A

Amazon Kinesis Data Analytics is the easiest way to analyze streaming data, gain actionable insights, and respond to your business and customer needs in real time.

Q23 Which AWS service provides managed robotics development?

✓ Correct Answer: A

AWS RoboMaker is a service that makes it easy to develop, simulate, and deploy intelligent robotics applications at scale.

Q24 What is the AWS service that provides managed mobile app development?

✓ Correct Answer: D

All these services support mobile app development: Amplify for full-stack development, Cognito for user authentication, and AppSync for GraphQL APIs.

Q25 What is the AWS service that provides managed GraphQL APIs?

✓ Correct Answer: B

AWS AppSync is a managed service that uses GraphQL to make it easy for applications to get exactly the data they need from multiple data sources.

Q26 What is the AWS service that provides managed time-series database?

✓ Correct Answer: C

Amazon Timestream is a fast, scalable, and serverless time series database service for IoT and operational applications that makes it easy to store and analyze trillions of events per day.

Q27 Which AWS service provides managed workflow orchestration?

✓ Correct Answer: C

AWS Step Functions is a serverless orchestration service that lets you combine AWS Lambda functions and other AWS services to build business-critical applications using visual workflows.

Q28 What is the AWS service that provides managed single sign-on?

✓ Correct Answer: C

AWS IAM Identity Center (successor to AWS Single Sign-On) helps you securely create or connect your workforce identities and manage their access centrally across AWS accounts and applications.

Q29 Which AWS service can be used to automate software deployments?

✓ Correct Answer: C

AWS CodeDeploy is a deployment service that automates application deployments to Amazon EC2 instances, on-premises instances, serverless Lambda functions, or Amazon ECS services.

Q30 Which Amazon EC2 instance pricing model can provide discounts of up to 90%?

✓ Correct Answer: D

Spot Instances offer the largest potential discount, often reaching up to 90% off the On-Demand price. This discount is contingent upon bidding for unused EC2 capacity, but instances can be terminated with a two-minute warning if the Spot price exceeds your bid.

Q31 Which AWS service provides managed container orchestration?

✓ Correct Answer: C

Amazon ECS (Elastic Container Service) is a fully managed container orchestration service that makes it easy to deploy, manage, and scale containerized applications using Docker containers.

Q32 Which AWS service provides managed key management?

✓ Correct Answer: B

AWS Key Management Service (KMS) makes it easy for you to create and manage cryptographic keys and control their use across a wide range of AWS services and in your applications.

Q33 Which AWS service provides managed secrets rotation?

✓ Correct Answer: A

AWS Secrets Manager provides managed secrets rotation capabilities, automatically rotating secrets like database credentials, API keys, and other sensitive information on a schedule you define.

Q34 A user has a stateful workload that will run on Amazon EC2 for the next 3 years. What is the MOST cost-effective pricing model for this workload?

✓ Correct Answer: B

Reserved Instances offer a significant discount (up to 75%) compared to On-Demand Instances in exchange for a commitment to use the EC2 instance for a 1-year or 3-year term. This is ideal for consistent, predictable workloads.

Q35 Which AWS service provides managed code repository?

✓ Correct Answer: A

AWS CodeCommit is a fully managed source control service that hosts secure Git-based repositories, making it easy for teams to collaborate on code in a secure and highly scalable ecosystem.

Q36 A user wants to allow applications running on an Amazon EC2 instance to make calls to other AWS services. The access granted must be secure. Which AWS service or feature should be used?

✓ Correct Answer: C

IAM roles are the recommended way to grant permissions to applications running on EC2 instances to access other AWS services. An IAM role allows an EC2 instance to assume a specific identity with defined permissions without embedding long-term credentials.

Q37 Which AWS service provides managed satellite communication?

✓ Correct Answer: A

AWS Ground Station is a fully managed service that lets you control satellite communications, process data, and scale your operations without having to worry about building or managing your own ground station infrastructure.

Q38 What is the primary benefit of using Amazon RDS instead of running a database on Amazon EC2?

✓ Correct Answer: C

Amazon RDS is a managed database service that handles routine database tasks such as provisioning, patching, backup, recovery, failure detection, and repair, reducing the administrative burden on users.

Q39 What is the AWS service that provides managed Apache Kafka?

✓ Correct Answer: B

Amazon MSK (Managed Streaming for Apache Kafka) is a fully managed service that makes it easy to build and run applications that use Apache Kafka to process streaming data.

Q40 Which AWS service provides DNS services?

✓ Correct Answer: B

Amazon Route 53 is a scalable and highly available Domain Name System (DNS) web service designed to route end users to Internet applications by translating domain names into IP addresses.

Q41 Which AWS service or feature can a company use to apply security rules to specific Amazon EC2 instances?

✓ Correct Answer: B

Security groups act as virtual firewalls for your EC2 instances, controlling inbound and outbound traffic at the instance level. They operate at the transport layer, filtering traffic based on protocols (TCP, UDP, ICMP), port numbers, and source/destination IP addresses. Each security group contains a set of rules that define the permitted network traffic.

Q42 A company runs many Amazon EC2 instances in its VPC. The company wants to use a native AWS security resource to control network traffic between certain EC2 instances. Which AWS service or feature will meet this requirement?

✓ Correct Answer: D

Security Groups are the most suitable native AWS service for controlling network traffic between specific EC2 instances within a VPC. They provide granular control at the instance level and use a stateful model for traffic filtering.

Q43 What is the AWS service that provides managed patch management?

✓ Correct Answer: B

AWS Systems Manager Patch Manager automates the process of patching managed instances with both security-related and other types of updates, helping you maintain compliance and security.

Q44 A company has multiple AWS accounts that include compute workloads that cannot be interrupted. The company wants to obtain billing discounts that are based on the company's use of AWS services. Which AWS feature or purchasing option will meet these requirements?

✓ Correct Answer: B

Consolidated billing allows a company with multiple AWS accounts to combine and pay for their total AWS usage under one bill. It aggregates usage across all linked accounts, making it easier to qualify for volume discounts on AWS services.

Q45 Which AWS service provides managed vulnerability assessment?

Q47 What is the AWS service that provides managed blockchain networks?

Q48 Which AWS service provides recommendations for optimizing AWS infrastructure costs?

Q49 Which AWS service allows for file sharing between multiple Amazon EC2 instances?
