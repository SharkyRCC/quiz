All these services support event-driven architecture: EventBridge for event routing, Lambda for serverless compute, and SNS for pub/sub messaging.

Q11 What is the AWS service that provides speech-to-text capabilities?

Q12 A company wants to save costs by archiving data that is no longer frequently accessed by end users. Which Amazon S3 feature will meet this requirement?

✓ Correct Answer: B

S3 Lifecycle policies automate the process of transitioning objects between different storage classes in S3 based on pre-defined rules. These rules can be based on object age or date, allowing for automated data archiving to lower-cost storage classes like S3 Glacier.

Q13 Which AWS design principle emphasizes the reduction of interdependencies between components of an application?

✓ Correct Answer: B

Loose coupling is the AWS design principle that specifically addresses the reduction of interdependencies between application components. It ensures components are independent and interact through well-defined interfaces.

Q18 A company wants to track tags, buckets, and prefixes for its Amazon S3 objects. Which S3 feature will meet this requirement?

Q22 Which S3 storage class is most cost-effective for data with unknown access patterns?

✓ Correct Answer: D

S3 Intelligent-Tiering is designed to automatically optimize storage costs by moving data between access tiers based on changing access patterns, making it ideal for unknown access patterns.

Q23 What does the Amazon S3 Intelligent-Tiering storage class offer?

✓ Correct Answer: C

Amazon S3 Intelligent-Tiering automatically optimizes storage costs by moving data between frequent, infrequent, and archive access tiers without operational overhead based on changing access patterns.

Q27 A company must archive its documents by using a write-once, read-many (WORM) model to meet legal and compliance obligations. Which feature of Amazon S3 can the company use to meet this requirement?

✓ Correct Answer: C

S3 Glacier Vault Lock is specifically designed to enforce WORM compliance. Once a Vault Lock policy is implemented, it prevents anyone, including the root user, from deleting or modifying the data in the vault according to the specified policy.

Q36 A company is storing data that will not be frequently accessed in the AWS Cloud. If the company needs to access the data, the data needs to be retrieved within 12 hours. The company wants a solution that is cost-effective for storage costs for each gigabyte. Which Amazon S3 storage class will meet these requirements?

✓ Correct Answer: B

S3 Glacier Flexible Retrieval is designed for long-term archive data where occasional access is needed. It provides retrieval times ranging from a few minutes to 12 hours and is considerably cheaper than other storage classes for infrequently accessed data.
