Q1 Which VPC component can a company use to set up a virtual firewall at the Amazon EC2 instance level?

✓ Correct Answer: B

Security groups act as virtual firewalls for EC2 instances, controlling inbound and outbound traffic at the instance level. They are stateful and provide granular control over traffic to individual instances.

Q2 What is the difference between a public subnet and a private subnet in a VPC?

✓ Correct Answer: B

A public subnet has a route to an internet gateway in its route table, allowing instances with public IP addresses to communicate with the internet. Private subnets don't have this route.

Q3 What is the AWS service that provides ultra-low latency for mobile applications?

✓ Correct Answer: B

AWS Wavelength brings AWS services to the edge of the 5G network, minimizing the latency to connect to an application from a mobile device.

Q4 Which AWS service provides continuous integration and deployment?

✓ Correct Answer: D

AWS CodePipeline is a fully managed continuous delivery service that helps you automate your release pipelines for fast and reliable application and infrastructure updates.

Q5 Which AWS service provides hybrid cloud storage?

✓ Correct Answer: C

AWS Storage Gateway is a hybrid cloud storage service that connects an on-premises software appliance with cloud-based storage to provide seamless and secure integration.

Q6 What is the AWS service that provides workflow orchestration?

✓ Correct Answer: D

AWS Step Functions is a serverless orchestration service that lets you combine AWS Lambda functions and other AWS services to build business-critical applications.

Q7 What is the AWS service that provides API call logging?

✓ Correct Answer: C

Q8 A developer needs to interact with AWS by using the AWS CLI. Which security feature or AWS service must be provisioned in the developer's account to meet this requirement?

✓ Correct Answer: D

To interact with AWS services programmatically through the AWS CLI, you need AWS access keys (access key ID and secret access key) for authentication.

Q9 Which AWS service provides load balancing capabilities?

✓ Correct Answer: C

Elastic Load Balancing automatically distributes incoming application traffic across multiple targets, such as Amazon EC2 instances, containers, and IP addresses.

Q10 What is the AWS service that provides managed Kubernetes?

✓ Correct Answer: B

Q11 What is the AWS service that provides managed Apache Kafka?

✓ Correct Answer: B

Q12 What is the AWS service that provides infrastructure as code using Chef or Puppet?

Q13 Which AWS service provides centralized logging?

Q15 Which AWS service provides real-time data streaming?

Q16 What is the AWS service that provides intelligent search?

Q17 What is the AWS service that provides pub/sub messaging?

Q19 What is the AWS service that provides general AWS service health information?

Q22 A company has an Amazon EC2 instance in a private subnet. The company wants to initiate a connection to the internet to pull operating system updates while preventing traffic from the internet from accessing the EC2 instance. Which AWS managed service allows this?

✓ Correct Answer: B

A NAT gateway allows instances in a private subnet to initiate outbound connections to the internet while preventing the internet from initiating inbound connections to those instances. This is perfect for downloading OS updates while maintaining security.

Q28 Which AWS service provides application performance monitoring?

Q29 What is the purpose of having an internet gateway within a VPC?

✓ Correct Answer: B

An Internet Gateway enables instances within the VPC to connect to the internet and allows traffic from the internet to reach these instances. It provides a target in route tables for internet-routable traffic and performs NAT for instances with public IP addresses.

Q31 What is the AWS service that provides DDoS protection?

Q32 Which AWS service provides configuration management?

Q33 What is the AWS service that provides infrastructure monitoring?

Q34 Which of the following can be components of a VPC in the AWS Cloud? (Choose two.)

✓ Correct Answer: D,E

Internet gateways and subnets are fundamental components of a VPC. An internet gateway allows communication between the VPC and the internet, while subnets are ranges of IP addresses within the VPC where you can launch AWS resources.

✓ Correct Answer: D
